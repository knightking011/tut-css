## FLEXBOX 
 It is one of the major component in css .it is used to make and align a layout for the website .use to align the details in our website more clearly and conveniently is called as FLEXBOX 

 ````
 .container{
    display:flex;
    gap:10px ; /* it is used to fix the gap between the text*/
 }
 ````

 inline flexbox is used to arrange the elements in the required space not the full page or line
