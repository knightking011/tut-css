# DIRECTION IN FLEX 

in this we can able to congfigure whether the flexbox is need to in row wise or column wise for the respective web page
        
          we use 'flex-direction : row or column;' to define 
example :
````
.navigation{
    background-colour:aqua;
    border: 10px solid red;
    display:inline-flex;
    flex-direction:row;
    gap:10px;
}
````
