# float property

so float property is basically used to wrap or float the image to any side like right or left and the surrounding test will get arranged to that 
````css

.img{
    float:right;
}

.img2{
    float:left;
}

````

## Clear tag

it is basically used for footer to go below the float bars like bottom of the page 
````
footer{
     background-colour:red;
     clear:both ; 
     /* both means it comes under both images we can manipulate it to whether right or left*/ 
}
````
