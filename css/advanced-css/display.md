# CSS  display 

it is used for displaying & configure the line & properties to choose that whether the block needs to be in same line or next line

````css
.red{
    display:inline-block; /* it states the block is inline and can arrange in the same line*/
    height:200px;
    width:200px; 
}

.green{
    display:block;/* it fullfills the whole line*/
    height:200px;
    width:200px;
}
````
